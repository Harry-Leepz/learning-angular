[ngStyle]

ngStyle is a ATTRIBUTE directive that can be used to change the style property of an element. It takes a JavaScript object with colon seperated key value pair, key being the style property, and the value being an expression.

Example:

The paragraph element below will either be colored Red or Green depending of whether the number randomly generated with the Math.random() is greater than 0.5. The result of the getColor() method is assigned using the [ngStyle] directive

Text color will be green or red

[ngClass]

ngStyle is a ATTRIBUTE directive that can be used to change the CSS class dynamically of an element. It takes a JavaScript object with colon seperated key value pair, key being the CSS class name property, and the value being an expression.

Example:

The paragraph element below will have the .green or .red CSS class applied, depending of whether the number randomly generated with the Math.random() is greater than 0.5 and the serverStatus is set as 'online'

Text color will be green or red depending on whether the serverStatus in 'online' or 'offline'
